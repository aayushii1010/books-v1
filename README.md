# 📚 Booksy – Your Personal AI Book Matcher

Booksy is a Python-based book recommendation system that helps readers discover books based on their interests, moods, and reading preferences. Whether you're looking for a cozy comfort read, an exciting thriller, an inspiring non-fiction book, or an adventurous sci-fi journey, Booksy provides personalized recommendations in seconds.

## ✨ Features

* 📖 Fiction and Non-Fiction categories
* 🎭 Multiple reading vibes and genres
* 🎲 Randomized book recommendations
* 📚 Up to 3 recommendations at a time
* 📝 Recommendation history tracking
* 🖥️ Interactive menu-driven interface
* 😊 Beginner-friendly and easy to use

## 🚀 How It Works

1. Launch the program.
2. Choose between Fiction and Non-Fiction.
3. Select your preferred reading vibe.
4. Receive personalized book recommendations.
5. View your recommendation history anytime.
6. Continue exploring books or exit the application.
